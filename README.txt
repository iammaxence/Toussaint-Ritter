Comment exécuter le binaire ?
- Double click sur le fichier "Executable.jar"

Où se situe la base de test de qualite ?
-La base de test ce situe dans le fichier "samples". La base de test est celle de Varoumas

Où se situe la base de test de performance ?
-Elle peut être généré via le code source. (Attention ! Le fichier fera environ 3Go de données).
	Voici les étapes:
	1.Deziper l'archive zip "ProjetAlgavFin" pour récuprer les fichiers sources
	2.Aller dans le package "test"
	3.Dans la class java "FilePointsGenerator.java", click droit, run.

Où se situe les test et les graphes utilisés pour le rapport?
-Dans le dossier GraphFiles.

Où se stiue le code source ?
-Dans l'archive zip nommé "ProjetAlgavFin"


Informations suplémentaires: Les fichier crée sont des fichiers d'extention ".csv". Pour les graphe, j'ai utiliser excel d'où les extentions ".xlsx" pour les fichiers contenu dans le dossier GraphFiles.
